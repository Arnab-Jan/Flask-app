Python Neural Network on Flask

This project is a simple web application that hosts a neural network, built from scratch in Python with numpy, to solve the classic XOR problem. The web interface is built with Flask.

How to Run Locally

Create a virtual environment:

python -m venv venv


Activate it:

Mac/Linux: source venv/bin/activate

Windows: .\venv\Scripts\activate

Install packages:

pip install -r requirements.txt


Run the app:

python app.py


Open http://127.0.0.1:5000 in your browser.

Running on GitHub Codespaces

On your GitHub repo page, click the green <> Code button.

Go to the "Codespaces" tab and click "Create codespace on main".

Once the Codespace loads, run the commands from the "How to Run Locally" section in the Codespace terminal.

Codespaces will automatically detect the running app and give you a public URL to test it.
